Descripción General

Implementación educativa completa de blockchain desde cero, incluyendo: minería con algoritmo Proof-of-Work, validación de cadena, transacciones seguras y API REST.

Este proyecto demuestra comprensión profunda de conceptos de criptografía, estructuras de datos, consenso distribuido y programación de alto rendimiento.

Características Principales

1. Algoritmo Proof-of-Work

Minería real con dificultad ajustable, hash SHA-256, y validación de nonce. Tiempo de minería estimado configurable.

2. Validación de Cadena Completa

Verificación de integridad de toda la cadena, validación de hashes, detección de manipulación y forks.

3. Sistema de Transacciones

Creación, firmado digital y validación de transacciones. Pool de memoria para transacciones pendientes.

4. Criptografía RSA

Llaves pública/privada para firma digital de transacciones. Verificación criptográfica de autenticidad.

5. API REST Completa

Endpoints para crear bloques, transacciones, validar cadena, consultar balance y sincronizar nodos.

6. Visualización Dashboard

Panel web interactivo para visualizar la cadena, transacciones pendientes, estadísticas de minería en tiempo real.

Arquitectura del Blockchain

Estructura de Bloque

class Block:
    - index: número secuencial del bloque
    - timestamp: marca de tiempo
    - transactions: lista de transacciones
    - previous_hash: hash del bloque anterior
    - nonce: número para Proof-of-Work
    - hash: hash SHA-256 del bloque actual
    - difficulty: nivel de dificultad de minería

Estructura de Transacción

class Transaction:
    - sender: dirección pública del remitente
    - receiver: dirección pública del receptor
    - amount: cantidad de criptomoneda
    - timestamp: marca de tiempo
    - signature: firma digital RSA

Instalación y Uso

Configuración

$ cd blockchain-python
$ python -m venv venv
$ source venv/bin/activate
