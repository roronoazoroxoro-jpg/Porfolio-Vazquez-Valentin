Descripción General

Asistente inteligente basado en IA que actúa como sistema de vigilancia personal para el hogar o lugar de trabajo. Detecta situaciones de emergencia a través de audio y video, visualiza el estado del usuario y puede activar automáticamente alertas al 911.

Este proyecto combina Computer Vision, procesamiento de lenguaje natural, síntesis de voz y análisis de patrones de comportamiento.

Características Principales

1. Reconocimiento de Voz Natural

Procesamiento continuo de audio a través de Google Speech-to-Text. Comprensión de comandos complejos e integración con NLP.

2. Detección de Palabras Clave de Emergencia

Sistema entrenado para detectar palabras clave como "ayuda", "peligro", "enfermo", "accidente". Activación automática de protocolo de emergencia.

3. Análisis de Gestos y Movimiento

Usa OpenCV y MediaPipe para detectar caídas, posturas anormales o inactividad prolongada. Sistema de alertas inteligente.

4. Interfaz Visual Dashboard

Panel que muestra estado del usuario en tiempo real, historial de actividad, alertas activas y configuración de sensibilidad.

5. Integración con Servicios de Emergencia

Llamada automática al 911 con envío de ubicación GPS, información médica del usuario y confirmación de emergencia.

6. Respuestas de Voz Personalizada

Síntesis de texto a voz para confirmaciones, alertas y comunicación bidireccional con el usuario o familiares.

Arquitectura del Sistema

Flujo de Detección de Emergencia

1. Audio Input → Speech Recognition
2. NLP Analysis → Detección de palabras clave
3. Confidence Score > Threshold?
   - SI: Video Analysis (verificación visual)
   - NO: Mantener escucha
4. Video Verification → Detección de caída/actividad
5. Alert Confirmed → Contactar 911
6. Sistema notifica a contactos de emergencia

Módulo Auditivo

Captura de audio, Speech-to-Text, análisis de palabras clave.

Módulo Visual

Captura de video, detección de personas, análisis de gestos y caídas.

Motor IA

NLP, clasificación de emergencias, toma de decisiones.

Sistema de Respuesta

Síntesis de voz, notificaciones, integración con 911.

Instalación y Configuración

Requisitos

- Python 3.9+
- Micrófono y cámara web
- Google Cloud API Key
- Acceso a servicios de emergencia (911)
- Conexión a internet estable

Instalación

$ git clone https://github.com/vazquez-valentin/jarvis-assistant
$ cd jarvis-assistant
$ pip install -r requirements.txt
$ python configure_api_keys.py
$ python jarvis.py

Configuración de Emergencia

# config.json
{
    "emergency_keywords": ["ayuda", "peligro", "911"],
    "confidence_threshold": 0.85,
    "emergency_contact": "+1-911",
    "backup_contacts": ["+1-XXX-XXX-XXXX"],
    "location_enabled": true,
    "auto_call_enabled": true
}
