Descripción General

Sistema de detección facial de última generación integrado con autenticación biométrica de múltiples capas. Utiliza redes neuronales convolucionales (CNN) para la detección en tiempo real con una precisión del 99.5%.

La aplicación está diseñada para sistemas de seguridad empresarial, permitiendo control de acceso sin contacto, registro automático de asistencia con marcas de tiempo y alertas de seguridad en tiempo real.

Características Principales

1. Detección Facial en Tiempo Real

Procesa múltiples rostros simultáneamente a 30 FPS utilizando algoritmos optimizados con CUDA para aceleración GPU.

2. Sistema Multi-Factor de Autenticación

Combinación de reconocimiento facial + contraseña biométrica + verificación de liveness (antiespoofi ng) con video profundo.

3. Encriptación End-to-End

Los datos biométricos se almacenan encriptados con AES-256. Tokens JWT con expiración para todas las sesiones.

4. Dashboard de Administración

Panel de control con estadísticas en tiempo real, historial de accesos, reportes de intentos fallidos y gestión de usuarios.

5. Integración con Cámaras IP

Compatible con cámaras RTSP y sistemas DVR. Soporta múltiples flujos de video simultáneamente con balanceo de carga.

6. Análisis de Seguridad Avanzado

Detección de comportamientos sospechosos, alertas de merodeadores, registro de eventos anómalos en base de datos.

Arquitectura Técnica

Frontend

Dashboard web con React.js, visualización de cámaras en tiempo real con WebRTC.

Backend

API REST en Flask + Gunicorn, procesamiento asincrónico con Celery, cache Redis.

Procesamiento

Python + OpenCV + TensorFlow, GPU CUDA compatible, procesamiento distribuido.

Base de Datos

PostgreSQL con extensiones de encriptación, backup automático en AWS S3.

Instalación y Uso

Requisitos

Python 3.9+
OpenCV 4.5+
TensorFlow 2.10+
PostgreSQL 12+
GPU NVIDIA (recomendado)

Instalación

$ git clone https://github.com/vazquez-valentin/facial-recognition
$ cd facial-recognition
$ python -m venv venv
$ source venv/bin/activate  # en Windows: venv\Scripts\activate
$ pip install -r requirements.txt
$ python setup_db.py
$ python app.py

Configuración

# .env file
DATABASE_URL=postgresql://user:pass@localhost/facial_db
JWT_SECRET=your-secret-key
CAMERA_RTSP_URL=rtsp://camera-ip:554/stream
GPU_ENABLED=true
MAX_CONCURRENT_STREAMS=5
