Descripción General

Plataforma de transporte compartido completamente funcional, comparable a Uber. Implementa matching de pasajeros con conductores en tiempo real, GPS tracking, sistema de calificaciones y pagos.

Este proyecto demuestra habilidades avanzadas en desarrollo full stack, arquitectura escalable, comunicación en tiempo real y aplicaciones geolocalización.

Características Principales

1. Matching Inteligente

Algoritmo de matching que conecta pasajeros con conductores cercanos. Considera distancia, dirección y disponibilidad.

2. GPS en Tiempo Real

Tracking de conductor durante el viaje. Visualización de ruta optimizada con Google Maps, actualización cada segundo.

3. Sistema de Pagos

Integración con Stripe. Cálculo de tarifa dinámico basado en distancia, tiempo y demanda. Múltiples métodos de pago.

4. Calificaciones y Reseñas

Sistema de rating bidireccional. Historial de viajes con detalles, reseñas de usuarios, perfil de conductor/pasajero con estadísticas.

5. Chat en Tiempo Real

Chat integrado entre pasajero y conductor usando Socket.io. Notificaciones push para mensajes nuevos y actualizaciones de viaje.

6. Panel de Conductor

Dashboard para conductores con estadísticas de ganancias, solicitudes entrantes, mapa interactivo, historial de viajes y gestión de disponibilidad.

Arquitectura Técnica

Frontend (React)

SPA con Redux para estado global, Google Maps SDK, notificaciones en tiempo real.

Backend (Node.js)

API REST con Express, Socket.io para WebSocket, autenticación JWT.

Base de Datos

MongoDB con esquemas de Usuarios, Viajes, Transacciones y Calificaciones.

Integraciones

Google Maps API, Stripe Payments, Twilio SMS, Firebase Notifications.

Instalación y Uso

Preparación

$ git clone https://github.com/vazquez-valentin/uber-platform
$ cd uber-platform

# Variables de entorno necesarias
$ cp .env.example .env
# Agregar: GOOGLE_MAPS_API_KEY, STRIPE_KEY, MONGODB_URI

Instalación Backend

$ cd backend
$ npm install
$ npm run dev

# API disponible en http://localhost:5000

Instalación Frontend

$ cd frontend
$ npm install
$ npm start

# Aplicación en http://localhost:3000
