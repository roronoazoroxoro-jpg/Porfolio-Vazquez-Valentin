📋 Paso 1: Abre el Portafolio

Hay dos formas de hacerlo:

Opción A: Doble click en index.html

Simplemente haz doble click en index.html en la carpeta del portafolio.

Opción B: Servidor Local (Recomendado)

Abre terminal/cmd en la carpeta y ejecuta:

python -m http.server 8000

Luego abre en navegador: http://localhost:8000

🎨 Paso 2: Personaliza Tu Información

Edita estos archivos:

config.json - Datos personales y contacto

{ "personalInfo": { "nombre": "TU NOMBRE AQUI", "email": "tu.email@example.com" } }

index.html - Contenido principal

Busca y cambia:

Vazquez Valentin → Tu Nombre

css/styles.css - Colores (opcional)

--primary-color: #00ff41; /* Verde neon */ --secondary-color: #ff0099; /* Magenta */

🎓 Paso 4: Personaliza Proyectos

Para editar un proyecto existente:

1. Abre: projects/facial-recognition.html

2. Cambia el título, descripción y contenido

3. Guarda cambios

Para añadir un nuevo proyecto:

1. Copia cualquier archivo de proyecto

2. Renómbralo: projects/tu-proyecto.html

3. Edita contenido

4. Añade tarjeta en index.html

🌐 Paso 5: Publica en Línea

Opción A: GitHub Pages (Gratuito)

git init git add . git commit -m "Mi portafolio" git push -u origin main

Luego activa GitHub Pages en Settings

Opción B: Netlify (Muy Fácil)

1. Ve a netlify.com

2. Arrastra la carpeta del portafolio

3. ¡Listo! Tu sitio estará en línea

Opción C: Vercel (Súper Rápido)

1. Ve a vercel.com

2. Conecta tu repositorio GitHub

3. Deployment automático
